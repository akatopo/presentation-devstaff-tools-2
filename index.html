Key sequences

"Compose key" for and

karabiner-elements.pqrs.orgwincompose.info

Getting this to work on MacOS

Install karabiner-elements

brew cask install karabiner-elements

Remap right_option to non_us_backslash

Edit your ~/Library/KeyBindings/DefaultKeyBinding.dict accordingly

{
  "§" = {
    "." = {
        "." = ("insertText:", "…");
    };
    "+" = {
        "1" = ("insertText:", "👍");
    };
  };
}

Restart apps and have fun 👌🏻

Spaceship

A zsh prompt for astronauts

denysdovhan.com/spaceship-prompt

Installing

  • zsh is required

  • a font with powerline glyphs is needed, fira code is good to go

  • install via npm

    npm install -g spaceship-prompt
  • more installation methods exist (ex. oh-my-zsh), check website
